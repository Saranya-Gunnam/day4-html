VS CODE

Visual Studio Code features a lightning fast source code editor, perfect for day-to-day use. With support for hundreds of languages, VS Code helps you be instantly productive with syntax highlighting, bracket-matching, auto-indentation, box-selection, snippets, and more

FEATURES

  • Editing: A fast editor with features like syntax highlighting, bracket-matching, and auto-indentation
  • Debugging: Debug your code without leaving the editor
  • Version control: Built-in support for Git and other source control providers
  • Testing: Run automated tests and view test coverage
  • Extensions: A marketplace of extensions to boost productivity
  • Configurability: Many options to configure the editor, including hiding or toggling parts of the user interface

EMMET KEYS

symbols description
! Expands to the complete html5 template
> Used to create child elements within a parent element
+ Used to create sibling elements
^ Moves up one level in the hierarchy
* Create multiple elements
{} for content
[] for attributes

Shortcut Keys

shortcut key description
Ctrl+S Save the current file
Ctrl+C Copy the selected text or item
Ctrl+V Paste the copied content
Ctrl+X Cut the selected text or item
Ctrl+Z Undo the last action
Ctrl+Y Redo the previously undone action
Ctrl+A Select all content in the current file
Ctrl++ Zoom in the editor view
Ctrl+- Zoom out the editor view
